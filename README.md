***Playwright Project***

This is an end-to-end testing project built using Playwright. It includes automated test scenarios, reusable page objects, and a structured framework to ensure maintainability and scalability.

**Project Structure**
Playwright-Project/
│
├── tests/                
├── pages/                
├── fixtures/             
├── playwright.config.ts  
├── package.json          
└── README.md

**Installation**

Make sure you have Node.js installed.

Then clone the repository and install dependencies:

git clone https://github.com/Majharul-Munna/Playwright-Project.git
cd Playwright-Project
npm install


Install Playwright browsers:

npx playwright install

**Running Tests**

Run all tests:

npx playwright test

**Test Results (Reports)**

Generate and view HTML report:

npx playwright show-report


Playwright also stores:

/playwright-report
/test-results
